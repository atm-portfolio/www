Introduction

My name is Athila Matias, and I work in the tech industry mainly as a Frontend Engineer and Full Stack Developer using JavaScript and Node.js technologies to build robust and user-friendly web applications and services.

I'm a passionate developer with a strong background in building either from scratch or refactoring existing code to achieve the best possible results.

I have extensive experience in building web applications using React, Vue, Angular, JavaScript, Node.js, as many other modern frameworks and libraries to effectively collaborate on diverse teams and projects, I possess a deep understanding of the key factors that contribute to a project's success.

Project: C# Monolithic Platform into Microservices architecture (Jan 2011 - Jul 2016)

Participated in rebuilding a monolithic C# platform into a scalable microservices architecture with Angular as the main presentation framework, Node.js as the backend API, and Docker as the containerization solution.

Contributed to the development of a file processor specialized microservice, a critical component of the company's infrastructure, which is responsible for reading CSV files from partner banks and processing them into a reward system.

Developed and maintained websites with marketing campaigns, internal development tools and administration systems to optimize workflows, streamline data input/output, and ensure seamless interactions throughout the system.

Project: E-commerce React components (Jul 2016 - Aug 2017)

Designed, developed and maintained reusable React components for a large e-commerce sports company, ensuring consistency, scalability and reusability across multiple applications.

Created and executed a suite of comprehensive React and Koa.js integration tests, ensuring high quality and reliable deliveries.

Project: Modernization of a large-scale internationalization resource system (Sep 2017 - Jul 2018)

Modernized a monolithic JavaScript and Jade application into a Node.js middleware layer, which handles multi language text files and interfaces with a Java platform. Additionally, the view engine was migrated from Jade to Marko.js, resulting in a more maintainable and scalable solution.

Upon completion, I delivered a fully functional administration system, consisting of a dashboard, user management, and a file management interface. The system uploads files, processes them with custom business logic in the middleware, and sends the results to the Java platform for broadcast into their respective applications and other services.

Project: Extended warranty claims (Aug 2018 - Oct 2019)

Developed and maintained a suite of Angular components for a project that provides claim filing to an extended warranty system. The project involved the development of new features, as well as the refactoring of existing components to improve performance and maintainability. This involved working with a remote team to identify and prioritize tasks, as well as implementing testing and deployment scripts.

The project also involved the upgrade of the Angular version from Angular 2 to the latest version at the time. This involved the refactoring of existing code to use the new features and best practices of the latest Angular version.

All new and refactored components were delivered with comprehensive unit tests and documentation.

Project: Refactor and expansion of Web client Admin system and Vehicle Hybrid Client (Nov 2019 - Ongoing)

Recently, I was involved in conducting a comprehensive rewrite of an outdated Angular 1 project to React.

This complex web client interfaces with Kotlin as the server-side language. The application's features include generating complex PDF reports, rendering a Fleet Tracking subsystem that interfaces vehicle data with Google Maps, creating administration pages, controlling cameras, and logging events. Additionally, the server collects data from the vehicle client, which is then used to generate other specific reports such as driver logs and unidentified driving events.

To ensure compliance with government regulations, the application requires careful structuring and documentation.

The vehicle client uses Angular and runs as a hybrid Android app on tablets.

Project: Pricing Application for a company that sells mobiles, tablets and accessories (Jan 2021 - Mar 2024)

On this project, I was responsible for maintaining and creating React specific components for a web application that runs exclusively on Safari and is delivered as a native app using a custom Node.js middleware. This application is installed on devices in-store, allowing customers to interact with in-store equipment and see prices and customizations.

All components are thoroughly tested and documented using Playwright. The application is built to support multiple languages and is designed to render text resources dynamically.

Project: Performance Upgrades, Component migrations and Webpack Optimizations (Apr 2024 - Oct 2024)

In this project, I have optimized the performance of an e-commerce web store that sells makeup, skincare, haircare and cosmetic products. I also refactored some React components, using best practices and patterns to improve code maintainability and applied various Webpack optimizations to improve the overall performance of the application.

Other tasks included stabilizing layout shifts in the application with their Server Side Rendering (SSR) implementation and optimizing lazy loading of assets.

Project: Api and React components refactoring and testing (Nov 2024 - Dec 2024)

In this project, I refactored some React components to improve the maintainability and scalability of the codebase. Additionally, I integrated a comprehensive testing strategy for those components in their testing projects using Playwright, including end-to-end and unit tests.

The project involved working with law enforcement data, which required precise and reliable input data, documentation and testing.

Project: Github web interface maintenance (Jan 2025 - May 2025)

I rejoined the company I collaborated with from 2021 to 2024 for a new project, where I focused on maintaining and refactoring several Vue components. My responsibilities included validating and expanding numerous test scenarios using Playwright for comprehensive end-to-end testing and utilizing Jest for thorough unit testing.

In this project, I maintained a custom user interface built on top of GitHub API. This interface allows non-technical users to create and manage text and asset resources directly in GitHub repositories, without requiring knowledge of Git command-line operations. This interface is utilized by internal users and is a critical tool for them to do their job.

Additionally, I successfully upgraded the Node.js version from 14 to 20, ensuring compatibility with other internal NPM packages in their ecosystem.
